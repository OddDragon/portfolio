Core4 Concept Technical Definition Real-Life Analogy Python Example Scratch Example
Variables Stores data for the program in a way such that what it stores can changethroughout the run. Like a basket, storing fruit inside.
Loops A section of code that is repeated a set number of times. Like a person knowing that they need to knock on a door repeatedly to be let in the house.
Conditionals A statement so the computer knows to only run a section of code under specific circumstances. Like a cat only jumping on a table and knocking a water glass off when she is bored.
Functions A piece of code that performs an action. For example, a person flexing their muscles so they can run or lift something heavy.
